# Contribuer au wiki HeavenCube

Cette section s’adresse aux personnes qui souhaitent corriger une information, améliorer une explication ou documenter une fonctionnalité confirmée du serveur.

## Prérequis

- Disposer d’une information vérifiable.
- Lire le [guide de style](style-guide.md).
- Ne jamais transformer une supposition en information officielle.

## Commencer

1. Repérez la page concernée.
2. Vérifiez l’information auprès d’une source officielle.
3. Respectez la structure et le ton du wiki.
4. Prévisualisez le rendu Retype si l’outil est disponible.
5. Proposez la modification selon la procédure du dépôt.

[!button text="Guide de style" icon="typography"](style-guide.md)

## Conseils pratiques

!!!tip Une information utile est une information vérifiable
Indiquez précisément ce qui manque avec un champ entre crochets lorsque l’équipe doit encore fournir une valeur.
!!!

## Questions fréquentes

!!!question Puis-je documenter une commande vue sur un autre serveur ?
Non. Une commande ne doit être publiée que si elle a été confirmée pour HeavenCube.
!!!

!!!question Où placer une capture d’écran ?
Ajoutez uniquement une image fournie ou validée par le propriétaire dans `assets/images/`, puis suivez les recommandations du guide de style.
!!!
