# Aide et services HeavenCube

Cette section rassemble les pages qui vous permettent de suivre l’actualité du serveur, d’obtenir de l’aide et d’accéder aux services officiels.

## Par où commencer ?

[!card layout="compact" title="Discord" text="Suivre les annonces et contacter la communauté." icon="comment-discussion"](discord.md)

## Prérequis

- Utiliser uniquement les liens publiés par HeavenCube.
- Ne jamais transmettre votre mot de passe Minecraft ou Discord.
- Préparer votre pseudonyme et les détails utiles avant de contacter le support.

## Accéder aux services

1. Choisissez la page correspondant à votre besoin.
2. Vérifiez les informations encore indiquées entre crochets.
3. Suivez uniquement les liens confirmés par l’équipe.
4. En cas de doute, utilisez la page [Support](support.md).

## Conseils pratiques

!!!tip Sécurité
Vérifiez toujours le domaine du site avant de vous connecter ou d’effectuer un paiement.
!!!

## Questions fréquentes

!!!question Où trouver les annonces ?
Consultez la page [Discord](discord.md). Le lien officiel doit encore être validé dans la documentation.
!!!

!!!question Où signaler un problème ?
La page [Support](support.md) indique les informations à préparer et le canal à utiliser dès qu’il aura été confirmé.
!!!
