# Découvrir le mode Survie

Le mode Survie est l’espace où vous récoltez des ressources, développez une base et découvrez les systèmes de progression de HeavenCube. Cette page vous aide à choisir quoi faire en premier.

!!!warning Configuration à confirmer
Le moyen d’accéder à la Survie, les mondes disponibles et leurs règles précises doivent encore être renseignés par l’équipe.
!!!

## Prérequis

- [ ] Avoir lu les [Premiers pas](../premiers-pas.md).
- [ ] Connaître le [règlement](../reglements.md).
- [ ] Comprendre les [protections de terrain](claims.md) avant de déposer des objets importants.

## Accéder à la Survie

| Information | Valeur officielle |
| --- | --- |
| Accès au mode | `[PORTAIL, MENU OU COMMANDE D’ACCÈS À LA SURVIE]` |
| Mondes disponibles | `[LISTE DES MONDES DE SURVIE]` |
| Retour au point d’accueil | `[COMMANDE DE RETOUR AU POINT D’ACCUEIL]` |
| Téléportation vers une zone libre | `[COMMANDE OU MÉTHODE DE TÉLÉPORTATION VERS UNE ZONE LIBRE]` |

## Bien commencer

1. Rejoignez le mode avec la méthode officielle.
2. Repérez une zone libre sans empiéter sur une construction existante.
3. Récoltez les ressources nécessaires à un abri simple.
4. Protégez votre terrain avant d’y placer des coffres importants.
5. Choisissez ensuite une activité : quêtes, métiers, économie, pêche ou événements.

## À quoi servent les fonctionnalités ?

| Fonctionnalité | Utilité pour le joueur |
| --- | --- |
| [Protections de terrain](claims.md) | Empêcher les autres joueurs de modifier une zone sans votre autorisation. |
| [Grades](grades.md) | Comprendre les paliers de progression et leurs conditions lorsqu’ils seront confirmés. |
| [Quêtes](quetes.md) | Suivre des objectifs guidés et connaître les récompenses prévues. |
| [Métiers](metiers.md) | Faire progresser une activité régulière et consulter ses gains officiels. |
| [Économie](economie.md) | Utiliser la monnaie du serveur et échanger de manière sûre. |
| [Pwarps](pwarps.md) | Visiter les lieux publics créés par les joueurs. |
| [Pêche](peche.md) | Comprendre les éventuelles mécaniques de pêche personnalisées. |
| [Événements](evenements.md) | Participer aux animations annoncées par l’équipe. |

## Erreurs fréquentes

- Stocker des objets avant d’avoir protégé le terrain.
- Construire près d’une base sans vérifier les règles de distance.
- Utiliser une commande copiée depuis un autre serveur.
- Investir dans un système dont les coûts ou conditions ne sont pas encore confirmés.

## Questions fréquentes

!!!question Les mondes sont-ils remis à zéro ?
La politique officielle est `[POLITIQUE DE REMISE À ZÉRO DES MONDES]`.
!!!

!!!question Le combat entre joueurs est-il actif ?
Consultez `[RÈGLES ET ZONES DE COMBAT ENTRE JOUEURS]` avant toute confrontation.
!!!

!!!question Les fermes automatiques sont-elles autorisées ?
Les limites sont `[LIMITES DES FERMES ET MACHINES AUTOMATIQUES]`. Demandez confirmation avant de construire un système intensif.
!!!
